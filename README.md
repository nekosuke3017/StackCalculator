# StackCalculator v1.0.1 by ねこすけ

## コマンド
* /calc [1スタックあたりの数(64, 16, 1など)] [変換する数] : [変換する数]に入力した値をスタックと余りで表示します。
* /calchelp : StackCalculatorのヘルプを表示します。

## 使い方
Minecraft Server\plugins\ にコピーし、/plまたは/pluginsでコピーされたことを確認する。
